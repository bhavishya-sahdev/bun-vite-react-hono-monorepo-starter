# bun-vite-react-hono-monorepo-starter

To install dependencies:

```bash
bun install
```

To run client:

```bash
bun run dev:client
```

To run server:

```bash
bun run dev:server
```

This project was created using `bun init` in bun v1.0.12. [Bun](https://bun.sh) is a fast all-in-one JavaScript runtime.
